grid

Simple ES6 grid widget.
No external dependencies, just pure ES6 javascript.

**[DEMO](http://htmlpreview.github.io/?https://github.com/tborychowski/grid/blob/master/index.html)**

###Features
- flexible & easy to use
- sortable
- filterable
- themable
- tiny
